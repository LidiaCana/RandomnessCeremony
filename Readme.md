# How to launch

```
forge create --rpc-url RPCADDRESS --private-key PRIVATEKEYHERE src/RandomnessCeremony.sol:RandomnessCeremony
forge create --rpc-url RPCADDRESS --private-key PRIVATEKEYHERE  src/LottoCeremony.sol:LottoCeremony --constructor-args "0xRANDOMNESSCEREMONYADDRESS"
```
